## The problem
Running a business on WhatsApp means never logging off. Customers message at 2am asking the same questions, and a missed reply is a lost sale.

## The solution
Sahara answers for you — grounded in your catalog and uploaded docs, in Urdu, Roman Urdu, or English — and flags only the messages that need a human.

## ✨ What makes it different
| | |
|---|---|
| 🌐 **Multilingual** | Replies in Urdu, Roman Urdu and English — the way your customers actually type. |
| 📚 **Hybrid RAG** | Answers grounded in your catalog + docs (pgvector dense **+** keyword full-text, RRF-fused). |
| 📸 **Vision OCR** | Snap a photo of a price list — Sahara reads it and starts answering from it. |
| ⚡ **Semantic FAQ cache** | Repeat questions return a cached reply in <50ms — fewer LLM calls, lower cost. |
| 🙋 **Smart escalation** | Auto-answers the routine; routes the rest to you. |

## 🏗️ Tech
Next.js 15 · PostgreSQL + **pgvector** · Groq + Gemini (free-tier-first) · hybrid retrieval · vision OCR · per-plan message quotas · Vercel.
